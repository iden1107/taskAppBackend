# サービス概要
macOSのリマインダーライクなtodoアプリです。検索窓を設置し、設置期日を超過したものはアイコンで通知。タグはカラーリングでカテゴリーをわかりやすくしました。終了したタスクは毎月棒グラフで表示し、タグ別の円グラフで割合を表示する機能を追加しました。
